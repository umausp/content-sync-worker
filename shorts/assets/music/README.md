# Music beds (optional)

Drop a **royalty-free / CC0** audio file here (`.mp3`, `.m4a`, `.aac`, `.wav`, or `.ogg`)
and the renderer will automatically mix it UNDER the narration (ducked to ~10% gain,
loudness-normalized, looped/trimmed to the video length).

If this folder is empty, Shorts render **narration-only** — which is clean and
professional; many top news Shorts use no music bed.

Where to get free, commercial-safe beds:
- YouTube Audio Library (youtube.com/audiolibrary) — free, cleared for YouTube
- Pixabay Music (pixabay.com/music) — CC0
- Free Music Archive (freemusicarchive.org) — filter to CC0 / CC-BY

Pick something **subtle and low-energy** (ambient/lo-fi/underscore) so it doesn't
fight the voice. The renderer already keeps it quiet, but the source track matters.

DO NOT add copyrighted music — it will get the video demonetized or Content-ID claimed.
